# Obsidian Templates

Here are my Obsidian templates using [Templater](https://silentvoid13.github.io/Templater/)
or the regular [Obsidian Templates](https://help.obsidian.md/Plugins/Templates).  

**Work agenda**. These templates should be automatically applied by default in custom folders.  
- `Weekly template`. Title format: `Week 250515` (YYMMDD). It provides automatic links to the previous and following weeks, as well as to your Google Calendar. Requires Templater.  
- `Meeting Template`. Title format: `Meet 250515T1530` (YYMMDDTHHmm), rounding every 15 minutes. It provides an automatic link to the current week note. Requires Templater.  

Additionally:
- `Tasks.md`. Lists together all unfinished tasks from all previous meetings and weeks. Requires Dataview.  

**Book highlights**  
- `KOReader template`. Parse JSON highlights from KOReader into Obsidian notes. Requires Templater.  

**Diary**  
- `Daily template`. It provides the date as the title, and a mood selector using tags.  
- `Dream template`. Similar to the daily template, but for writing your dreams.  

Additionally:
- `Mood.md`. Creates a heatmap calendar of your mood across the years. Requires Dataview, Heatmap Calendar.  

**Other markdown templates**  
- `Table template`  
- `Quote template`  
- `Gantt template`
